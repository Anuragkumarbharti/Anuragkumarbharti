### 📖 Current Research: *Quantum Cryptography and Secure Key Exchange*

> Studying secure communication methods based on **Quantum Key Distribution (QKD)**, focusing on how quantum cryptographic techniques improve the security of key exchange mechanisms compared to classical approaches. The research involves understanding the **mathematical foundations of quantum cryptography** and analyzing communication protocols such as **BB84**.

**Key research activities:**
- Study of classical cryptographic key exchange mechanisms
- Analysis of the BB84 quantum key distribution protocol
- Comparison between classical and quantum cryptographic security models
- Security analysis of quantum communication systems

### ⚛️ [Quantum Key Distribution Simulation using Qiskit](https://github.com/Anuragkumarbharti/Quantum-Key-Distribution-Simulation-using-Qiskit)

<a href="https://github.com/Anuragkumarbharti/Quantum-Key-Distribution-Simulation-using-Qiskit">

> Full simulation of the **BB84 Quantum Key Distribution protocol** using IBM Qiskit. Includes qubit preparation, basis selection, measurement, intercept-resend eavesdropping attacks, and **Quantum Bit Error Rate (QBER)** analysis with visualization plots.

| Feature | Description |
|---|---|
| 🔑 BB84 Protocol | Complete implementation of qubit encoding, transmission & sifting |
| 🕵️ Eavesdropper Simulation | Intercept-resend attack modeling with Eve |
| 📊 QBER Analysis | Error rate computation with 11% security threshold detection |
| 📈 Visualizations | 5 publication-quality plots (basis comparison, key comparison, QBER sweep) |
